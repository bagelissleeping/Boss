# BOSS 求职助手

用于 BOSS 直聘岗位筛选、复核与批量投递的 Chrome 侧边栏插件。

插件把岗位列表识别、规则判断、批量处理进度和结果回看收敛到浏览器侧边栏，帮助求职者减少重复浏览和手动整理成本。

## 功能

- 识别 BOSS 直聘列表页岗位卡片，并提取公司、岗位、地点、薪资、经验、学历和标签等信息。
- 基于可配置规则判断岗位去向：推荐投递、待复核、直接跳过。
- 支持批量处理当前列表页岗位，并按轮次滚动读取更多岗位。
- 记录本轮结果和今日投递数量，减少重复处理。
- 提供规则中心，可编辑年限硬筛和目标关键词。
- 提供页面诊断能力，便于 BOSS 页面结构变化后排查问题。

## 技术栈

- Vue 3
- TypeScript
- Vite
- CRXJS / Chrome Manifest V3

## 安装与开发

```bash
npm install
npm run dev
```

开发时在 Chrome 扩展管理页打开“开发者模式”，加载本项目生成的扩展目录。

生产构建：

```bash
npm run build
```

构建产物位于 `dist/`，该目录不纳入 Git 管理。

## 使用方式

1. 打开 BOSS 直聘岗位列表页。
2. 点击浏览器扩展图标，打开侧边栏。
3. 确认页面状态为“已连接”。
4. 在规则中心调整年限硬筛、目标关键词等偏好。
5. 回到侧边栏执行“开始筛选与投递”。

## 注意事项

- 默认规则仅为通用示例，使用前请按自己的求职目标修改。
- 批量投递会操作真实网页，请先小范围验证页面识别和规则判断。
- BOSS 直聘页面结构变更后，可能需要调整解析选择器。
- 本项目仅用于个人求职效率工具和作品展示，请遵守目标网站规则与相关法律法规。
